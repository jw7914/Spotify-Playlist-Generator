# Spotify Playlist Generator

A full-stack application that leverages the Spotify API and Google's Gemini AI to generate custom playlists based on your listening history and preferences.

## Deployment Link
https://spotifyplaylistgen.vercel.app/

## Features

- **Spotify Integration**: Log in with your Spotify account to access your top tracks and artists along with other account information.
- **AI-Powered Recommendations**: Uses Google Gemini to analyze your taste and generate unique playlist concepts.
- **Smart Playlist Creation**: Create new playlists directly to your Spotify account.
- **Modern UI**: Built with React, Tailwind CSS, and HeroUI for a sleek, responsive experience.
- **Playback Control**: Control your Spotify playback directly from the app.

## Tech Stack

### Frontend
- **Framework**: React with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: HeroUI (NextUI)
- **Icons**: Lucide React

### Backend
- **Framework**: FastAPI (Python)
- **AI Model**: Google Gemini (via `google-genai`)
- **HTTP Client**: `httpx` / `requests`
